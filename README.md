# Dylan — Architecture & Design Portfolio

A minimal, responsive portfolio website for architecture and design work.

## Files
- `index.html` — main page
- `style.css` — all styling
- `script.js` — scroll animations
- `images/` — place your project photos here

## Images
Replace the placeholder images in the `images/` folder with your own:
- `hero.jpg` — homepage hero image
- `project1.jpg` — first project
- `project2.jpg` — second project
- `project3.jpg` — third project
- `studio.jpg` — about/studio photo

Recommended image size: 1400px wide, saved as high-quality JPG.
